The Code

The Javascript starts with the function getString which retrieves the user string input using documnet.getElementById

Once the string is retrieved the code continues the palindrome process by reversing the string with the reversedString function.

After the string is reversed, the new string is evaluated against the original to check to see if they are equal with the checkPalindrome function

The results are then returned to the page with the displayPalindrome fucntion.

To reverse a string with the reversedString function we first must take the input string and loop through each character one at a time. While looping through we must build the new string for later evaluation. To complete this, an empty array is created as newString. The for loop begins by using the string method for returning the length of the original string. Since a string can be evaluated as an array we take the length of the string and -1 which allows then loop to start at the last character. As the loop progresses it will continue to subtract 1 from the length until the index value is greater than or equal to 0. While each iteration happens the present charater is appened to then array. Once complet the new string is returned.

After the new reversed string is built, the newString is then variable is evaluated against the original input. The boolean result is then returned.

After all the previous functions are complete, the final result of the revered string and the boolean value on the palindrome status are returned to the dom and displayed with document.getElementById("message").innerHTML = ${finalString} A PALINDROME!`; if the word is indeed a palindrome. If the word is not a palindrome then document.getElementById("message").innerHTML = `${finalString} IS NOT A PALINDROME!`; is returned. 


                            import json


                            def reverse_string(string):
                                reversed_string_list = []
                                for char in string[::-1]:
                                    reversed_string_list.append(char)
                            
                                formatted_string = ''.join(reversed_string_list)
                            
                                return formatted_string
                                
                                
                            def evaluate_palindrome(string, reversed_string):
                                is_palindrome = True
                                
                                if string != reversed_string:
                                    is_palindrome = False
                                    return is_palindrome
                                else:
                                    return is_palindrome
                            
                            def lambda_handler(event, context):
                                
                                string = event['queryStringParameters']['request_string']
                                reversed_string = reverse_string(string)
                                is_palindrome = evaluate_palindrome(string, reversed_string)
                                
                                request_reponse = {}
                                request_reponse['statusCode'] = 200
                                request_reponse['headers'] = {}
                                request_reponse['headers']['Content-Type'] = 'application/json'
                                request_reponse['headers']['Access-Control-Allow-Origin'] = '*'
                                request_reponse['body'] = json.dumps(
                                    {'response': 
                                        {'reversedString': reversed_string, 
                                        'isPalindrome': is_palindrome}})
                            
                                return request_reponse
